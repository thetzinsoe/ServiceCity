# 08-03: Search Bar Redesign — Header Integration & Modern Styling

**Status:** Implemented
**Scope:** Dashboard (Bookings) + Customers + Drilldown pages + Booking Confirmation
**Date:** 2026-06-21

---

## Design Decisions

| Decision | Choice |
|----------|--------|
| Layout | Title+Count (left) · Search bar (right, flex-grow) |
| Count style | Inline muted text after title, e.g. `Bookings (42 total · 5 pending)` |
| Search trigger | Enter key only — no Search button |
| Search height | ~42px (taller than current `form-control-sm` ~31px) |
| Corners | Pill shape — `border-radius: 24px` |
| Mobile | Title+Count row 1, search bar full-width row 2 |
| Clear | "×" icon button inside the search bar (right side), shown when search active |

## Tasks

### Task 1: Add `.search-bar-modern` CSS class to `site.css`

New class with:
- `height: 42px`
- `border-radius: 24px` (pill shape)
- `padding-left: 2.5rem` (space for search icon)
- `padding-right: 2.5rem` (space for clear × button)
- Clean `border: 1px solid #dee2e6`
- Focus state using existing `--sc-blue` token
- Mobile: full-width

### Task 2: Update `Dashboard.cshtml` — header row

**Current (lines 6-25):**
```html
<div class="d-flex justify-content-between align-items-center mb-3">
    <h2>Bookings</h2>
    <span class="badge">42 total · 5 pending</span>
</div>
<form class="mb-4">...</form>
```

**Proposed:**
```html
<div class="d-flex justify-content-between align-items-center mb-3 gap-3 flex-wrap">
    <div class="d-flex align-items-baseline gap-2">
        <h2 class="h4 fw-bold mb-0">Bookings</h2>
        <span class="text-muted small">(@total total · @pendingCount pending)</span>
    </div>
    <form asp-action="Dashboard" method="get" class="flex-grow-1" style="max-width: 400px;">
        <div class="position-relative">
            <span class="position-absolute top-50 start-0 translate-middle-y ms-3 text-muted">🔍</span>
            <input type="text" name="search" value="@Model.Search"
                   class="form-control search-bar-modern"
                   placeholder="Search by phone, reference, or name…"
                   onkeydown="if(event.key==='Enter')this.form.submit()" />
            @if (!string.IsNullOrWhiteSpace(Model.Search))
            {
                <a asp-action="Dashboard"
                   class="position-absolute top-50 end-0 translate-middle-y me-2 btn btn-sm btn-link text-muted p-0"
                   title="Clear search">&times;</a>
            }
        </div>
    </form>
</div>
```

- Remove the standalone `<form>` and `<button type="submit">`
- Remove `form-control-sm` — use new `search-bar-modern` class
- Search icon (🔍) positioned absolutely inside the input
- Clear "×" link positioned absolutely on the right
- Form submits on Enter key
