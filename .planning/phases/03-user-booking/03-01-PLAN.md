### 3. Create Booking Controller

**File:** `ServiceCity/Controllers/BookingController.cs`

Actions:
- `Create` (GET) — Show booking form with service categories dropdown
- `Create` (POST) — Validate, generate reference number, save booking, redirect to confirmation
- `Confirmation` (GET) — Show confirmation page with reference number
- `Lookup` (GET) — Show phone lookup form
- `Lookup` (POST) — Find bookings by normalized phone, display results
- `Status` (GET) — Show booking status page by reference number

**Reference Number Generation:**
```csharp
private static string GenerateReferenceNumber()
{
    const string chars = "ABCDEFGHIJKLMNOPQRSTUVWXYZ0123456789";
    var random = new Random();
    var suffix = new string(Enumerable.Range(0, 8)
        .Select(_ => chars[random.Next(chars.Length)])
        .ToArray());
    return $"SC-{suffix}";
}
```

**Duplicate Prevention:**
- Generate IdempotencyKey from: CustomerPhoneNormalized + ServiceCategoryId + PreferredDate + hash of form data
- Check for existing booking with same IdempotencyKey before insert
- Use database unique constraint as safety net

**Rate Limiting:**
- Track booking submissions per phone number in last hour
- Reject if > 5 submissions with clear error message
